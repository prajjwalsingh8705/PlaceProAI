# PlacePro AI

Predict. Prepare. Get Placed.

## Overview

PlacePro AI is an AI-powered Placement Readiness Coach that helps students improve their chances of placement through:

- Resume Analysis
- Placement Prediction
- AI Mock Interviews
- Speech Analysis
- Posture Analysis
- Personalized Roadmaps
