# PlacePro AI Requirements

## Functional Requirements

### User Authentication

* User Registration
* User Login
* JWT Authentication
* Profile Management

### Resume Analyzer

* Upload PDF Resume
* Extract Skills
* Extract Projects
* Extract Certifications
* Generate Resume Score

### Placement Prediction

* Accept User Inputs
* Predict Placement Probability
* Generate Recommendations

### AI Interview Assistant

* Generate Interview Questions
* Generate Follow-Up Questions
* Evaluate Answers
* Store Interview Results

### Speech Analysis

* Convert Audio to Text
* Calculate Speaking Speed
* Detect Filler Words

### Posture Analysis

* Track Head Position
* Track Attention
* Generate Posture Score

### Roadmap Generator

* Generate 7-Day Plan
* Generate 30-Day Plan
* Generate 90-Day Plan

---

## Non-Functional Requirements

* Secure Authentication
* Fast API Response
* Scalable Architecture
* Mobile-Friendly UI
* Modular Design
* Easy Deployment
